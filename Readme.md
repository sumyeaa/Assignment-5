1.Difference between var, let, and const

var:
Can be changed anytime.
Can be used before declaring.
works inside a function.

let:
Can be changed, but cannot be used before declaring.
works only inside {} where it is declared.

const:
Cannot be changed after assigning.
Block-scoped like let.
Good for values that should stay same

2.Spread Operator(...)

it is called the spread operator.
It is used to copy arrays or objects or split items.

3.Difference between map(), filter(), and forEach()

map():
Creates a new array by changing each item.
Returns a new array.

filter():
Creates a new array with only items that pass a condition.

forEach():
Loops through each item without creating a new array.

5.Template Literals

Template literals let you write strings with variables easily.
Use backticks ` instead of quotes.
You can write multi-line strings and insert variables by ${}.
